## Core Features:
Admin :
+ Add order of goods.
+ View history of transport/order.
+ Live map.
+ Using GPS, keep track of multiple cars location.

Driver :
+ View history of transport per day.
+ Live map.
+ Notification to finalized transaction when arriving at delivery location.
